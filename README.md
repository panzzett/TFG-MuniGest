# TFG-MuniGest

Prototipo de plataforma de tramitación de expedientes electrónicos para Administraciones Locales.

Este proyecto ha sido desarrollado por Carlos Galvez Reguera como parte del Trabajo de Fin de Grado en Ingeniería Informática de la Universidad Internacional de La Rioja (UNIR).

## Descripción

La aplicación permite gestionar expedientes electrónicos, documentos asociados, estados de tramitación, firma electrónica simulada y registro de acciones para garantizar trazabilidad básica.

## Tecnologías previstas

- Python
- FastAPI
- SQLite
- SQLAlchemy
- HTML, CSS y JavaScript
